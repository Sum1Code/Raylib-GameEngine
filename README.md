## How to build
- `git clone https://github.com/Sum1Code/Raylib-GameEngine`
- `cd Raylib-GameEngine`
- `make`
### OR
- `git clone https://github.com/Sum1Code/Raylib-GameEngine`
- `cd Raylib-GameEngine`
- `mkdir build && cd build`
- `cmake ..`
- `make`
