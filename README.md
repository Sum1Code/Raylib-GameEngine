# Raylib-GameEngine !!STILL IN DEVELOPMENT!!

A Unity-Like Game engine made with Raylib.. cheating? definitely... 

## DEPENDENCIES
- raylib 4.5.0

## How to build
- `git clone https://github.com/Sum1Code/Raylib-GameEngine`
- `cd Raylib-GameEngine`
- `make`
